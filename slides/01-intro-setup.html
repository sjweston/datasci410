Why data science for psychology?

Psychology research involves:

  • Collecting data from experiments, surveys, observations
  • Cleaning messy real-world data
  • Visualizing patterns and relationships
  • Communicating findings clearly

Data science gives us tools to do all of this better.

The old way vs. the new way

Point-and-click (SPSS, Excel)

  • Hard to reproduce
  • Error-prone
  • Limited visualizations
  • Doesn‚Äôt scale

Code-based (R)

  • Fully reproducible
  • Transparent & shareable
  • Unlimited customization
  • Handles any data size

Why R specifically?

  • Free and open source ‚Äî no expensive licenses
  • Designed for statistics ‚Äî built by statisticians for data analysis
  • Massive community ‚Äî packages for everything
  • Psychology-friendly ‚Äî psych, lavaan, lme4, brms‚Ä¶
  • Beautiful visualizations ‚Äî ggplot2 is industry-leading
  • Reproducible research ‚Äî Quarto/RMarkdown integration

R Scripts

R scripts (.R files) are where you write and save your code.

To create one:

  1. File ‚Üí New File ‚Üí R Script
  2. Or press Ctrl/Cmd + Shift + N

Always save your scripts! The console history disappears.

Running code from a script

  • Run one line: Put cursor on line, press Ctrl/Cmd + Enter
  • Run selection: Highlight code, press Ctrl/Cmd + Enter
  • Run entire script: Ctrl/Cmd + Shift + Enter

Tip

You‚Äôll use Ctrl/Cmd + Enter constantly. Memorize it!

Why No AI? (Yet)

I encourage you to read ‚ÄúCognitive helmets for AI bicycles‚Äù by Cat Hicks.

  • Steve Jobs (1980): The computer is a ‚ÄúBicycle for the mind.‚Äù
    • It amplifies your pedaling.
    • You still provide the balance and direction.
  • Generative AI (2024): It‚Äôs more like a Motorcycle.
    • It provides the engine/speed.
    • If you don‚Äôt know how to ride, you crash faster and harder.

The Problem: The ‚ÄúIllusion of Competence‚Äù

  • Psychological Concept: Metacognition (Monitoring your own understanding).
  • The Trap:
    • AI gives you a working answer immediately.
    • It feels like you solved it (Dopamine hit!).
    • But you didn‚Äôt build the Mental Model of why it works.
  • The Result: When the AI hallucinates (which it will), you cannot debug it. You are stranded.

The Solution: ‚ÄúCognitive Helmets‚Äù

  • Before we get on the AI motorcycle, we need a Cognitive Helmet.
  • What is the Helmet?
    1. Syntax Literacy: Reading code as fluently as English.
    2. Debugging Resilience: Emotional regulation when things break.
    3. Strategic Friction: The ability to break a big problem into small pieces.
  • Source: Dr. Cat Hicks, Engineering Psychologist

Course Structure: Building the Helmet

  • Why so many short assignments?
    • This is ‚ÄúDesirable Difficulty.‚Äù
    • We are building the neural pathways for code syntax.
    • Small wins build the ‚ÄúMental Model‚Äù of how data is structured.
  • Why ‚ÄúMaladaptive Easy‚Äù?
    • Using AI now = Skipping the gym but expecting to get strong.
    • It bypasses the encoding process in your brain.

The Future: When Do We Use AI?

  • We will use AI in this field (and perhaps this course later).
  • But only when you have your Helmet.
  • The Goal: You should be the Pilot, not the Passenger.
    • You define the problem.
    • You verify the code.
    • You catch the errors.

Summary

‚ÄúIt is not about winning against another participant in an abstract lab task. It is about your real life, and your real life is a complex set of decisions‚Ä¶ Use the friction to build your strategy.‚Äù ‚Äî Dr. Cat Hicks
