Introduction to ggplot2

What is ggplot2?

  • Created by Hadley Wickham (2005)
  • Based on the Grammar of Graphics by Leland Wilkinson
  • Most popular R visualization package
  • Part of the tidyverse

The ‚Äúgg‚Äù stands for ‚ÄúGrammar of Graphics‚Äù

The grammar of graphics

Every ggplot has three essential components:

  1. Data ‚Äî what you want to visualize
  2. Aesthetics (aes) ‚Äî how variables map to visual properties
  3. Geoms ‚Äî what geometric shapes represent the data
# The basic template
ggplot(data = <DATA>, mapping = aes(<MAPPINGS>)) +
  <GEOM_FUNCTION>()

Breaking it down

ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) +
  # ‚Üë Data and aesthetic mappings
  geom_point()
  # ‚Üë Geometric object (points = scatterplot)
  • data = mpg ‚Äî use the mpg dataset
  • aes(x = displ, y = hwy) ‚Äî map displacement to x, highway mpg to y
  • geom_point() ‚Äî represent data as points

A cleaner way to write it

You can drop data = and mapping =:

# These are equivalent
ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + geom_point()

ggplot(mpg, aes(x = displ, y = hwy)) + geom_point()

ggplot(mpg, aes(displ, hwy)) + geom_point()  # x and y are first args

I‚Äôll use the middle version ‚Äî clear but not overly verbose.

Aesthetic mappings

What are aesthetics?

Aesthetics are visual properties of geoms:

  • x, y ‚Äî position
  • color ‚Äî outline color
  • fill ‚Äî interior color
  • size ‚Äî how big
  • shape ‚Äî what shape
  • alpha ‚Äî transparency

Mapping color to a variable

What if we want to see which points are which car class?

ggplot(mpg, aes(x = displ, y = hwy, color = class)) +
  geom_point()

Mapping color to a variable

Setting vs. mapping

Mapping ‚Äî aesthetic varies with data (inside aes())

ggplot(mpg, aes(x = displ, y = hwy, color = class)) +
  geom_point()

Setting ‚Äî aesthetic is constant (outside aes())

ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point(color = "blue", size = 3)

Setting aesthetics manually

ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point(color = "steelblue", size = 3, alpha = 0.6)

Setting aesthetics manually

Facets

What are facets?

Facets split your plot into small multiples based on a variable.

ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point() +
  facet_wrap(~class)

What are facets?

facet_wrap()

Creates a ribbon of panels:

# Syntax: facet_wrap(~variable)
ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point() +
  facet_wrap(~class)
  • Use ncol or nrow to control layout
  • scales = "free" allows different axis ranges

facet_grid()

Creates a grid of panels with two variables:

ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point() +
  facet_grid(drv ~ cyl)

facet_grid()

When to use facets

Facets are great when:

  • You have too many colors/shapes to distinguish
  • You want to compare patterns across groups
  • You want each group to ‚Äústand alone‚Äù

Saving plots

Use ggsave() to save your plot:

# Create the plot
my_plot <- ggplot(mpg, aes(x = displ, y = hwy, color = class)) +
  geom_point() +
  theme_minimal()

# Save it
ggsave("my_plot.png", my_plot, width = 8, height = 6)

Putting it together

A complete example

ggplot(mpg, aes(x = displ, y = hwy)) +
  geom_point(aes(color = class), size = 2, alpha = 0.7) +
  geom_smooth(method = "lm", color = "black", se = TRUE) +
  labs(
    title = "Fuel Efficiency Decreases with Engine Size",
    subtitle = "Data from 234 vehicles (1999-2008)",
    x = "Engine displacement (liters)",
    y = "Highway MPG",
    color = "Vehicle class",
    caption = "Source: EPA fuel economy data"
  ) +
  theme_minimal(base_size = 14)

Key takeaways

  1. Always visualize your data ‚Äî summary stats can hide patterns
  2. ggplot2 uses layers ‚Äî data + aesthetics + geoms
  3. Aesthetics can be mapped or set ‚Äî aes() vs direct values
  4. Facets are powerful ‚Äî split plots by categories
  5. Labels and themes matter ‚Äî make your plots readable
