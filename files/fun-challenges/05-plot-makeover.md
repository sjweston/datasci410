# Fun Challenge 5: Plot Makeover

**Due:** Sunday, May 3 at 11:59 PM

You've been given a plot (`05-plot-makeover.R`) that shows the relationship between sleep and stress, colored by exercise frequency. The plot uses the right geom, but it looks like a default ggplot --- no thought given to design.

---

## Your task

Improve the plot by changing **at least 5 things**. Ideas for what to fix:

- Add a clear, informative title
- Clean up axis labels (no underscores!)
- Switch to a better theme
- Choose more meaningful colors
- Add transparency to deal with overlapping points
- Fix the legend title
- Adjust point size
- Remove unnecessary gridlines

Submit:

1. Your improved R script
2. A screenshot of the new plot
3. A brief note (1--2 sentences per change) explaining what you changed and why
