Fun Challenge 4: Bug Hunt

Due: Sunday, April 26 at 11:59 PM

An R script (04-bug-hunt.R) is supposed to load a dataset of survey responses, clean it up, and make a plot of satisfaction by department. But it has 8 bugs. Find them all and fix them.

The bugs include things like:

  • Typos in function names
  • Missing or extra commas/parentheses
  • Wrong variable names
  • Logic errors
  • ggplot mistakes
  • Missing arguments

Your task

  1. Download 04-bug-hunt.R and 04-bug-hunt-data.csv and put them in the same folder
  2. Update the file path in the script to match your setup (e.g., "04-bug-hunt-data.csv")
  3. Try to run the script — it will break
  4. Find and fix all 8 bugs
  5. Submit the corrected script and a list of what you fixed

Submit: Your corrected .R file and a brief list describing all 8 fixes. One submission per team.
